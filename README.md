# 📱 My Collections App

Мобильное приложение на Flutter для хранения пользовательских списков: игр, фильмов, книг и секретной информации. Все данные сохраняются локально на устройстве.

---

## 🚀 Возможности

### 🧩 Главный экран
При запуске приложения отображаются 4 раздела:
- 🎮 Games
- 🎬 Films
- 📚 Books
- 🔐 Secrets

---

## 🎮 Games
- Отображение списка игр (сохраняются локально)
- Добавление новой игры
- Редактирование записи
- Удаление записи

**Форма:**
- `Name` (обязательное поле)
- `Studio` (необязательное поле)

---

## 🎬 Films
- Отображение списка фильмов
- Добавление / редактирование / удаление

**Форма:**
- `Name` (обязательное)
- `Director` (необязательное)

---

## 📚 Books
- Отображение списка книг
- Добавление / редактирование / удаление

**Форма:**
- `Name` (обязательное)
- `Author` (необязательное)

---

## 🔐 Secrets
- Защищённый раздел
- При входе требуется пароль
- Хранение чувствительных данных (например, пароли от сервисов)

**Форма:**
- `Resource` (обязательное)
- `Password` (обязательное)

---

## 💾 Хранение данных
- Используется локальное хранилище (`SharedPreferences`)
- Все данные сохраняются на устройстве пользователя
- Не требуется интернет-соединение

---

## 🛠️ Технологии
- Flutter
- Dart
- SharedPreferences
